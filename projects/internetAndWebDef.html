Internet

A connected set of computers/networks that allows us to roam, search and share with other networks and computers.

Web

A bunch of data that is stored and collected, we can access that data with it.

How are they different?

The internet is the connection of all the networks, while the web is an application to allow users to access data on the internet

ASCII and Unicode

Both are languages that computers use to encode data.

UTF-8

The most common character encoding for Unicode.

URL

An address to access something thats on the internet.

Domain Name

A user friendly way to locate something on the internet.

HTTP

A call and response protcol, meaning when searched for data it will find it and return it.

HTTPS

A form of security for the HTTP.

IP Address

A label of numbers assigned to each network for communication.

DNS

A process that converts domain names into their IP addresses so the can be used inside the computer.

W3C Recommendations

A set of rules/guidlines that are agreed upon by all internet users to uphold standard.

XML, XHTML, HTML, and HTML5

All languages that have similar foundations but have specific purposed in depth of each one.
